Usage: python caesar.py [options] [command] [message]

Commands:
    help                            Shows this message
    encrypt                         Encrypts the given message
    decrypt                         Decrypts the given message

Options:
    -p, --password PASSWORD         Uses PASSWORD for the en-/decryption
    -f, --file FILE                 En-/Decrypts FILE
    -o, --output FILE               Writes the en-/decrypted message in FILE
